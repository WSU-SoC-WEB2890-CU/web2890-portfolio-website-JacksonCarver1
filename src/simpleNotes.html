Simple Notes Page.

Just a simple notes page for a simple task.

The Details:


Using a bit of CSS and plenty of JavaScript + React, I was able to make a web page that works as a notes app. You can add a note to it and delete it just by tapping it! It also happens to have a few extra features such as counting the characters of the words in the note. This was a project we had to create in my WEB 3350 class, it only had the initial requirements of taking notes & quotes but I felt like it could have more features. This quick website was also just a challenge to only be created with JavaScript and React along with some CSS to make it look more appealing.
