All About Gaming Platforms!

A website that's all about the game platforms.

The Details:


This project was another class assignment at Weber State University, this one in particular was for WEB 2620. The project required the use of Sass, CSS, and html to make a fully-custom website that I designed myself. This was the first project that I did a markup and prototype for, I really enjoyed the process of making the design then implementing it in the actual code.


The part I enjoyed most about the project was choosing all the ui designs such as the colors, the buttons, the different fonts, etc. Although there is a few things I would change, I really love how it turned out. The main style chosen was to get a retro vibe yet stick to a more simple feel, which the website pulls off very well.
