My First Portfolio!

My first steps to becoming a web designer!

The Details:


This website was my first gateway into web design where I created my first portfolio. This was also one of the first websites I ever uploaded to a server! This was originally a project for my previous class at the WSU! (cs 2350). The purpose of this assignment was to get used to making a place where I can make my own brand and show off my skills as a web designer (in 2021 that is). We were required to get some feedback and all the feedback I got was very positive regarding the overall design of the website. There was definitely a few errors I made that I now know of, but that was my first portfolio...


My current coding expertise is waaay beyond what it was just a few years ago. The results of that website are still pretty amazing considering I had so little experience when I first started.
