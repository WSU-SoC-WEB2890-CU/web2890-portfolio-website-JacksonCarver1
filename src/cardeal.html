Cardeal Website

A one-page website designed for a car dealership company.

The Details:


This one-page website was a design that I created for an assignment in WEB 2890 at Weber State University. The whole point of this website was to use my skills that I have gained from previous classes and implement them into an actual website. The business it was based around was a car dealership that sells cars online which was meant to target online customers who didn't want to purchase a car in person.

The Method:


This one-page website was a design that I created for an assignment in WEB 2890 at Weber State University. The End Result:


The website included features such as a testimonial, a place where you could enter the car information, and many intuitive designs. With the use of CSS and a little bit of JavaScript, I was able to create a website with a great color scheme, design, and classic look. The feedback I got from this website was pretty great, I changed a few things such as the footer and a couple sections and it turned out perfect!
